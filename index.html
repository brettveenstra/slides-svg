Don’t Panic

Essentials

A vector image format 


<img src="foo.svg" alt="…">
          

.foo {
  background-image: url("foo.svg");
  /* etc. */
}

Acts like an image would (nothing fancy)

An XML-based document format 


<object data="foo.svg" type="image/svg+xml">
  <!-- Fallback content -->
</object>

Interactive, accessible, cacheable, animatable, scriptable

A subset of HTML5 


<p>
  There's an inline SVG right after this paragraph.
</p>

<svg>
  <!-- etc. -->
</svg>

Able to inherit in-page styles (currentColor and a lot more)

</svg>

The Future

CSS Shapes

W3C Candidate Recommendation (March 2014)


.float {
  shape-outside: polygon(10px 10px, 90px 50px, 40px 50px, 90px 90px, 10px 90px);
  shape-margin: 10px;
}
  • Supported in Chrome, Safari and Opera
  • Under Consideration for Edge

CSS Masking

W3C Candidate Recommendation (August 2014)


.masked {
  clip-path: polygon(15px 99px, 30px 87px, 65px 99px, 85px 55px, 122px 57px, 184px 73px, 198px 105px, 199px 150px, 145px 159px, 155px 139px, 126px 120px, 112px 138px, 80px 128px, 39px 126px, 24px 104px);
